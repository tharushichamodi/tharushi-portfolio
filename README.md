# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/tharushichamodi/tharushi-portfolio.git
```

Install packages

```bash
npm install
```

Run locally

```bash
npm run dev
```

Production Build

```bash
npm run build
```
